## 🖥️ What this app does

line-harness-oss is an open-source CRM for LINE Official Account users. It helps you manage chats, send step-based messages, build forms, set rich menus, score contacts, and automate common tasks.

Use it to keep customer work in one place instead of moving between tools. It is built for teams that want a free option for LINE account work.

## ✅ What you can do

- Set up step delivery flows

- Keep daily account work organized

## 📌 Main parts of the app

### 💬 Contact management

Keep user records in one place. You can store names, tags, notes, and activity history. This helps you follow each contact without losing track.

### 📣 Broadcasts

Send one message to many users at once. Use broadcasts for updates, reminders, and campaign messages.

### 🪜 Step delivery

Create a set of messages that go out one after another. This is useful for welcome flows, follow-ups, and simple automated journeys.

### 📝 Forms

Build forms to collect replies, signups, feedback, or lead details. Use forms to gather clean data without manual entry.

### 🎛️ Rich menus

Set up rich menus for quick actions inside LINE chat. These menus help users find key options fast.

### ⭐ Scoring

Assign points to contacts based on actions. Scoring helps you find active users and follow up with the right people.

### ⚙️ Automation

Set rules that handle repeat tasks for you. You can use automation for tags, follow-ups, and basic message logic.

## 🔒 Safe use tips
